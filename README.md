# School Management System

A simple static frontend for a School Management System built with HTML, CSS, and JavaScript. This repository currently contains the UI pages, styles, and client-side scripts.

This website is the equivalent of "Moodle.com" but for middle schools instead of University students.

**Status:** Frontend scaffold complete; backend to be made in the second half of the project.

**Built with:** HTML, CSS, JavaScript

**Pages / Key files**
- `Index.html`: Landing / home page — [Index.html](Index.html)
- `About.html`: About the school — [About.html](About.html)
- `Assignments.html`: Assignments view — [Assignments.html](Assignments.html)
- `Contact.html`: Contact page — [Contact.html](Contact.html)
- `Dashboard.html`: Logged-in dashboard — [Dashboard.html](Dashboard.html)
- `Login.html`: Login page — [Login.html](Login.html)
- `Register.html`: Registration page — [Register.html](Register.html)
- `Teacher-Profile.html`: Teacher profile view — [Teacher-Profile.html](Teacher-Profile.html)

Front-end assets:
- CSS: [CSS/](CSS/)
- JavaScript: [JS/](JS/)
- Images: [Images/](Images/)

**Quick start**
- Open `Index.html` in a browser, or serve the folder with a static server for hot reload (for example `Live Server` in VS Code).

**What’s implemented**
- Static pages for the main flows: home, about, contact, assignments, login/register, dashboard, and teacher profile.
- Responsive styles organized under `CSS/` and client logic in `JS/`.
